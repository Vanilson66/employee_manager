## Instalação

### Frontend

1. Vá para o diretório do frontend:
    ```bash
    cd frontend
    ```

2. Instale as dependências:
    ```bash
    npm install
    ```

3. Execute a aplicação:
    ```bash
    npm run dev
    ```

### Backend

1. Vá para o diretório do backend:
    ```bash
    cd backend
    ```

2. Instale as dependências:
    ```bash
    npm install
    ```

3. Execute o servidor:
    ```bash
    npm start
    ```

## Uso

Acesse `http://localhost:3000` para visualizar o dashboard.
